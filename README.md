# Financial Data Analysis using SQL

## Project Overview
This project analyzes financial transaction data using SQL to understand customer behavior, transaction trends, and financial activity.

## Tools Used
- SQL (MySQL)
- Data Analysis
- Finance Analytics

## Dataset Tables
- Customers
- Accounts
- Transactions

## Key Analysis Performed
- Total transaction amount per customer
- City-wise financial activity
- Credit vs debit transaction comparison
- Top 5 highest transactions
- Monthly transaction trends
- Customers with multiple accounts
- Average transaction per account

## Key Insights
- High-value transactions are concentrated among a small group of customers.
- Debit transactions exceed credit transactions indicating high spending behavior.
- Transaction activity peaks during certain months.

## Outcome
The project demonstrates how SQL can transform raw financial data into actionable business insights for better decision-making.
